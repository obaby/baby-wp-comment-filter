一个强大的WordPress评论过滤插件，支持字数限制、中文检测、关键词过滤等功能。

== Description ==

Baby WP 评论强化拦截插件是一个功能强大的WordPress评论过滤插件，旨在帮助网站管理员更好地管理评论内容，提高评论质量。

= 主要功能 =

* **字数限制**: 设置评论的最少和最多字数要求
* **中文检测**: 要求评论必须包含中文字符，过滤纯英文或数字评论
* **关键词过滤**: 支持自定义关键词和WordPress讨论设置的关键词
* **正则表达式支持**: 支持使用正则表达式进行高级匹配
* **错误消息自定义**: 可以自定义各种错误提示消息
* **WordPress集成**: 与WordPress后台讨论设置完美集成
* **测试功能**: 提供完整的测试页面方便调试

= 技术特性 =

* 简单架构设计，易于维护
* 高效的过滤算法，不影响网站性能
* 支持WordPress 3.0+版本
* 支持多语言环境
* 完善的输入验证和安全性

= 使用场景 =

* 个人博客评论管理
* 企业网站评论过滤
* 论坛评论质量控制
* 防止垃圾评论和广告

== Installation ==

1. 下载插件文件
2. 将插件文件夹上传到 `/wp-content/plugins/` 目录
3. 在WordPress后台"插件"页面激活插件
4. 进入"设置" → "Baby WP 评论"进行配置

== Frequently Asked Questions ==

= 插件会影响网站性能吗？ =

不会。插件采用高效的过滤算法，只在评论提交时进行必要的检查，不会影响网站的正常访问速度。

= 如何添加更多自定义关键词？ =

在插件设置页面的"自定义禁止关键词"文本框中，每行添加一个关键词即可。

= 支持正则表达式吗？ =

支持。在自定义关键词中使用 `/pattern/flags` 格式即可使用正则表达式。

= 如何恢复默认设置？ =

停用插件后重新激活即可恢复默认设置。
